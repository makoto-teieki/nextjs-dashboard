# Claude Codeへの指示テンプレート

このファイルには、新しいセッションでClaude Codeに効果的に指示するためのテンプレートが含まれています。

## 🚀 新しいセッション開始時

### 基本テンプレート

```
.claude/README.mdを参照して、このプロジェクトの全体像とドキュメント構造を把握してください。
その後、LEARNING_PROGRESS.mdで現在の学習進捗を確認してください。
```

### 学習継続時

```
LEARNING_PROGRESS.mdを確認して、前回どこまで学習したか教えてください。
その後、次のチャプターに進みたいです。
```

### エラー対応時

```
DEVELOPMENT.mdのエラーハンドリング手順を参照して、
[具体的なエラー内容]
このエラーを解決してください。
```

### 開発環境セットアップ時

```
DEVELOPMENT.mdを参照して、開発環境のセットアップ手順を実行してください。
```

## 💡 効果的な指示のコツ

### ✅ Good

```
「LEARNING_PROGRESS.mdで進捗を確認し、Chapter 14から再開してください」
```
→ **明確**: どのドキュメント、どのチャプターか指定

```
「DEVELOPMENT.mdの型チェック手順に従って、現在のコードをチェックしてください」
```
→ **具体的**: どのセクション、何をするか明確

### ❌ Bad

```
「前回の続きから」
```
→ **曖昧**: どこから？どのドキュメントを見る？

```
「エラーを直して」
```
→ **不明確**: どのエラー？どの手順に従う？

## 📋 状況別テンプレート

### 1. 新機能実装時

```
LEARNING_PROGRESS.mdで現在の理解度を確認してください。
[機能名]を実装したいです。関連する概念について説明してから実装してください。
```

### 2. コードレビュー依頼時

```
DEVELOPMENT.mdのベストプラクティスに従っているか、
[ファイルパス]のコードをレビューしてください。
```

### 3. デバッグ時

```
以下のエラーが発生しました：
[エラーメッセージ]

DEVELOPMENT.mdのトラブルシューティングセクションを参照して解決してください。
```

### 4. 学習の振り返り時

```
LEARNING_PROGRESS.mdを見て、これまでに学んだ内容を要約してください。
特に理解度が低い（⭐3以下）トピックについて、復習プランを提案してください。
```

## 🎯 ドキュメント更新の依頼

### 進捗更新

```
Chapter [番号]を完了したので、LEARNING_PROGRESS.mdを更新してください。
学んだことは以下です：
- [箇条書き]
```

### ベストプラクティス追加

```
今日発見した[問題/解決策]をDEVELOPMENT.mdの
「よくあるエラーと解決方法」セクションに追加してください。
```

## 🔄 定期的な確認

### 週次レビュー

```
LEARNING_PROGRESS.mdを確認して、今週の学習進捗をまとめてください。
来週の学習計画も提案してください。
```

### 月次振り返り

```
LEARNING_PROGRESS.mdの全体を見て、
- 達成した目標
- 理解度が向上したトピック
- 次の月の重点項目
を教えてください。
```

## 📚 複数ドキュメント参照時

```
以下のドキュメントを参照して[タスク]を実行してください：
1. LEARNING_PROGRESS.md - 現在の理解度確認
2. DEVELOPMENT.md - 実装ガイドライン
3. [他のドキュメント] - [目的]
```

---

**注意**: Claudeは各セッションが独立しているため、毎回明示的にドキュメントを参照するよう指示することが重要です。
