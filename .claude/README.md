## 📝 ドキュメント追加のルール

新しいドキュメントを追加する際は：

1. **このREADME.mdを更新**して一覧に追加
2. **明確な目的**を記載
3. **更新タイミング**を明記
4. **Claude参照タイミング**を明記

### テンプレート

```markdown
### [カテゴリ]
- **[ファイル名.md](./ファイル名.md)**
  - 説明
  - **更新タイミング**: いつ更新するか
  - **Claude参照**: いつ参照すべきか
```

## 🎯 ベストプラクティス

### ✅ Good
- 目的が明確なファイル名（`LEARNING_PROGRESS.md` > `notes.md`）
- このREADME.mdに必ず登録
- ファイル内に更新日時を記載

### ❌ Bad
- 曖昧なファイル名（`memo.md`, `temp.md`）
- インデックスに登録しない
- 古い情報のまま放置
